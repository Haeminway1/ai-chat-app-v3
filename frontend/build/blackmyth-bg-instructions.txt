Please add the background image file provided to this directory with the name "blackmyth-bg.jpg".

This image will be used as the background for the Black Myth theme. Make sure it has good quality and resolution, since it will be displayed in full screen as a background.

1. Temporarily, the app will use the image you provided in your message
2. For permanent use, replace the placeholder with your preferred image 
3. The image should be high resolution (1920x1080 px or higher) 
4. The image should have good contrast to ensure text readability
5. File format should be JPG for better compression

After adding the image, the Black Myth theme will automatically use it as the background.
